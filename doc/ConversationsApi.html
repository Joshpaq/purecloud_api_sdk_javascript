createConversationParticipantsByParticipantIdReplace(conversationId, participantId, userId, address, username, queueId, voicemail)

Replace this participant with the specified user and/or address

Parameters:
Name Type Description
conversationId string conversation ID
participantId string participant ID
userId string The user that will replace this participant. If address is not supplied then the user's Work address will be used. This parameter is required when replacing a participant that has an active chat.
address string The address that will be used to contact the new participant
username string The username of the person that will replace this participant. This field is only used if the userId is blank.
queueId string The id of the queue that will replace this participant.
voicemail boolean Indicates this participant will be replaced by the voicemail inbox of the participant.
